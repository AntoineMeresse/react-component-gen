# react-component-gen

The goal of this project is to create a component folder if it does not exist.
Then using param to create a component.js and component.css

# Add it to your bash/zsh

- ```code ~/.zshrc```
- ```alias createComponent="python3 path/react.py"```
- ```source ~/.zshrc```

# How to use it ?

From the terminal, in your src folder : ```createComponent Test```
